CSS 用于控制网页的样式和布局。CSS3 是最新的 CSS 标准

  1. CSS选择器(楼楼此处挑重点写)
    1. 类选择器、ID选择器、通配选择器、元素选择器
      1. 类选择器：在css样式中以“.”(点开头)后跟类名。(例：.demo{color:red;}表示html中所有class类叫做demo的字体颜色为红色)
      2. ID选择器：在css样式中以“#”(点开头)后跟ID名。(例：#demo{color:red;}表示html中ID叫做demo的字体颜色为红色)(注意：id唯一)
      3. 通配选择器：在css样式中以“*”(点开头)后跟花括号。(例：*{padding：0px;}表示html中所有元素的内边距为0)
      4. 元素选择器：在css样式中以标签名开头后跟花括号(例：p{color:red;}表示html中所有p标签的字体颜色为红色)
      5. 组合选择器：在css样式中多个标签拥有共同样式时可用，来减少代码量,每个标签间用标签隔开(例：h1,p{color:red;}表示html中所有p和h1标签的字体颜色为红色)
    2. 子选择器、伪元素(较多，不详细说明，请看例子)
      1. 例：div>p{ background-color:yellow;} 解释：选取父元素是 div 元素的每个 p元素,并设置其背景色为黄色。
      2. 例：div+p{ background-color:yellow;} 解释：选择 div元素之后紧跟的每个 p 元素并设置其背景色为黄色。
      3. 例：div p{ background-color:yellow;} (注意:这里div与p之间是空格) 解释：选择并设置位于 div 元素内部的每个 p 元素的样式
      4. 例：div>p{ background-color:yellow;} 解释：选取父元素是 div 元素的每个 p元素,并设置其背景色为黄色。
      5. 例：a[target=_blank]{ background-color:yellow;} 解释：为 target="_blank" 的 a 元素设置样式
      6. 例：[lang|=en]{ background-color:yellow;}解释：选择 lang 属性值以 "en" 开头的元素，并设置其样式
      7. 例：div[class^="test"]{background:#ffff00;}解释：设置 class 属性值以 "test" 开头的所有 div 元素的背景色
      8. 例：p:before{ content:"台词：";}(注意这里的:before为伪元素)解释：在每个 p 元素的内容之前插入新内容
      9. :nth-child(n) 选择器匹配属于其父元素的第 N 个子元素，不论元素的类型。n 可以是数字、关键词或公式。
        例：Odd 和 even 是可用于匹配下标是奇数或偶数的子元素的关键词（第一个子元素的下标是 1）。
        在这里，我们为奇数和偶数 p 元素指定两种不同的背景色：
        p:nth-child(odd){background:#ff0000;}
        p:nth-child(even){background:#0000ff;}

      3. 属性 描述
      transition 简写属性，用于在一个属性中设置四个过渡属性。
      transition-property 规定应用过渡的 CSS 属性的名称。
      transition-duration 定义过渡效果花费的时间。默认是 0。
      transition-timing-function 规定过渡效果的时间曲线。默认是 "ease"。
      transition-delay 规定过渡效果何时开始。默认是 0。
