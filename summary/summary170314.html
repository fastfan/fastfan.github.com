对象和数组

对象：就是一种类型，即“引用类型”，而对象的值就是引用类型的实例
在ECMAScript中引用类型是一种数据结构，用于将数据和功能
组织在一起，它常被称作“类”。
虽然ECMAScript是一门面向对象的语言，却不具备传统面向对象语言所支持的类和接口等基本结构。

  1. Object类型
    1. new运算符
    2. 字面量表示法

    2. 使用new运算符

    例：var goods=new Object();

    1. 为对象添加属性：通过对象名称+“.”+属性名来进行声明
      例：goods.name="西瓜"；
    2. 输出有俩种形式的写法
      1. document.write(goods['price'])
      2. document.write(goods.price)
    3. 为对象添加行为（函数或方法）
      例：goods.input();调用对象行为（函数）。
    4. 字面量创建属性和行为
      例：var user={ name:'xiaohei',
      sex:'男',
      age:18,
      music:function(){
      alert('我会唱歌');
      }
      } document.write(user.name+user['sex']);
      user.music();
      注意：属性名后用冒号而不是等于，属性之间用逗号而不是分号来相隔。
    5. 在实际开发中，更偏向于“字面量形式”表示。
    6. 遍历对象for in
      例：var user={ name='xiaohei',
      age=18,
      sex='男',}
      for (attr in user){
      document.write(user[attr]);
      }
      }zhuyi：attr接受的是string类型的属性名称。
