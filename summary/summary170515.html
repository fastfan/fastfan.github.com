JS复习(一)

  1. javascript简介
    1. Javascript诞生于1995年，最初的目的是处理以前由服务器端语言负责的一些输入验证操作。Javascript是一门弱类型语言。
    2. 一个完整的Javascript实现由下列三个不同的部分组成：
      1.核心(ECMAscript)
      2.文档对象模型(DOM)
      3.浏览器对象模型(BOM)
    3. ECMAscript规定了这门语言的“语法”，“类型”，“语句”，“关键字”，“保留字”，“操作符”，“对象”。等组成部分，并对实现该标准规定的各个方面内容的语言的描述。
    4. 文档对象模型DOM，是针对XML但经过扩展用于HTML的应用程序编程接口(API).DOM把整个页面映射为一个多层节点结构。HTML或XML页面中的每个组成部分都是某种类型的节点。
    5. 浏览器对象模型BOM，提供与浏览器交互的方法和接口。
  2. 在HTML中使用Javascript
    1. 在HTML页面中插入Javscript的主要方法，就是使用“script”元素。
      1. 内部嵌入：
        \<\script type="text/javascript"\>
        function sayHi(){
        alert("Hi");
        }
        <\/script\>
      2. 外部引入：
        \<\script type="text/javascript" src="xxx.js"\><\/script>
      3. 按照传统做法\<\script>\元素都应该放到head元素中，这种做法目的是要把所有外部文件的引用都放在相同的地方，但是如果需要的外部引用文件太多的话，就会导致浏览器呈现页面时出现延迟，所以，为了避免这个问题，现代web应用程序一般会把全部Javascript引用放到body元素中页面的后面。
  3. 基本概念
    1. 语法
      ECMAscript中一切（函数名、变量、操作符）都区分大小写；
      标识符：就是指变量、函数、属性的名字。规则：
      (1).第一个字符必须是一个字母、下划线(_)、或一个美元符号（$）
      (2).其他字符可以是字母、下划线(_)、数字或一个美元符号
    2. 注释："//"双斜杠表示单行注释，"/**/"表示多行注释。
    3. 语句：var sum=a+b; 以“;”分号结尾，虽然省略分号也是有效的语句，但是我们不推荐使用，因为加上分号会让我们避免很多错误。
    4. 关键字：用于表示控制语句的开始或结束，注意:关键字不可当标识符！
    5. 变量：用var操作符，后面跟变量名来定义.例:var a=0;注意:用var操作符定义的变量将成为定义该变量的作用域中的局部变量。即在函数中用var定义了一个变量，这个变量在函数退出后就会被销毁
    6. 数据类型：5种简单数据类型(基本数据类型):Undefined、String、Number、Boolen、Null。
    7. 操作符：，-，*，/,+=,-=,*=,%=,/=,!=,>,<,>,>=,<=
    8. 语句：条件语句:if,if...else
      循环语句:for,do...while,while语句

      function functionName(ag1,arg2,...,argN){//小括号内为参数
      statements//花括号内为函数体

  6. 匿名函数 和闭包(注意：总结以实例为主，下面直接上例子)

    7. 闭包就是能够读取其他函数内部变量的函数。由于在javascript中，只有函数内部的子函数才能读取局部变量，所以说，闭包以简单理解成“定义在一个函数内部的函数“。 所以，在本质上，闭包是将函数内部和函数外部连接起来的桥梁。

    1. function box(){
      return 'lee';//普通函数
      }
      alert(box());//lee
      function (){
      return 'lee';//单独匿名函数不可运行，就算能运行也无法调用，因为没有名称
      }
      把匿名函数赋值给变量就可运行
      var box=function (){
      return 'lee';
      };
      alert(box());
      通过自我执行来执行匿名函数
      (function (){
      alert('lee');//匿名函数（）（）；第一个括号放匿名函数，第二个圆括号执行
      })();
      自我执行的匿名函数的传参
      (function(age){
      alert(age);//100
      })(100);
      函数里放匿名函数
      function box(){
      return function(){
      return 'lee';
      }
      }
      alert(box()());//lee
      通过闭包返回局部变量
      function box(){
      var age=100;
      return function(){
      return age;
      }
      }
      alert(box()());//100
      使用全局变量进行累加
      var age = 100;
      function box(){
      age++;
      }
      alert(age);
      box();
      alert(age);
      使用局部变量进行累加
      function box(){
      var age=100;
      age++;
      return age;
      }
      alert(box());//101
      alert(box());//102
      alert(box());//103
      使用匿名函数实现局部变量驻留内存中从而累加
      function box(){
      var age=100;
      return function(){
      age++;
      return age;
      };
      }
      var b = box();
      alert(b());//101
      alert(b());//102
      alert(b());//103
