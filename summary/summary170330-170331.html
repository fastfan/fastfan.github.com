Git、Github上传文件

  1. 在码云上用GIT上传自己的代码
    1. 上传
      1. 先注册码云账号，完成后登陆找到新建项目，创建一个项目，完成后返回桌面
      2. 在本地合适的位置创建一个新的文件夹，叫test(名字随意)
      3. 进入文件夹，右键选择Git bash here
      4. git version ------ 获取当前版本信息
      5. git help ------ 帮助：得到主要的git命令
      6. git init ------ 初始化git仓库
      7. 编写帮助文件

    touch README.md(创建帮助文件)

    git add README.md (添加帮助文件)

    git commit -m "注释内容"（添加注释）
      8. 如果第一次安装git这时会提示下面：
  
  git  config --global user.name "用户名"  （这里的用户名是你要设置的git的全局姓名或昵称）
  
  git  config --global user.email "邮箱地址" （设置的全局邮箱地址）
    2. 然后再执行git commit -m "注释内容"（添加注释）就可以了
    3. 如何修改码云上的文件
      1. 在选择的文件夹中首先 git bash,输入git init 初始本地化环境
      2. 克隆GIT库代码到本地 git clone 网址
      3. cd 项目文件
      4. git add . （将所有文件和文件夹提交到待提交队列中）
      5. git commit -m "注释"
      6. git push origin master
  2. 在Github上用GIT上传自己的代码
    1. 首先注册Github的账号（为了减少不必要的麻烦邮箱号要用英文邮箱，QQ的英文邮箱就可以）
    2. 注册好之后登陆，你会发现Github和码云很相似，只不过Github全英文罢了
      接下来来的步骤同Git上传步骤一样（包括修改文件等）
  3. 建立网站
    1. 申请域名
      1. 用浏览器搜索“万网”，进入官网注册账号
      2. 搜索你想要的域名（这里是收费的）
      3. 选中你想要的域名（未被注册过的）付费操作
      4. 成功付费后打开你的控制台，找到你刚注册的域名进行解析
      5. 解析有俩种，第一种新手模式，另一种高级设置模式，新手模式系统会帮你搞定
        高级设置模式需自己配置
      6. 高级设置模式配置步骤： 记录类型：选NAME 主机记录：@ 完成再次创建一个解析 记录类型：选NAME 主机记录：www
      7. 在github打开项目新建一个文件，把你的域名复制进去
      8. 然后你就可以通过www.域名访问你的项目了
